30 days of Python programming challenge is a step-by-step guide to learn the Python programming language in 30 days!

## Installation

1. **Clone this repository:**

```bash
git clone https://github.com/mateolafalce/30-days-python.git
cd hacker5news
```

2. **Compile & Run**

```bash
cargo run --release
```

3. **Look at localhost**

```
http://localhost:5001
```
